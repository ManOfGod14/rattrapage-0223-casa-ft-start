L'API BOM et l'interface Window en JS

BOM : Browser Object Model

Les objects du BOM :

  • L'objet Navigator : qui représente l'état et l'identité du navigateur qu'on va utiliser avec l'API Geolocation
  • L'objet History : nous permet de manipuler l'historique de navigation du navigateur
  • L'objet Location : nous fournit les informations relatives à l'URL de la page courante
  • L'objet Screen : va nous permettre d'examiner les propriétés de l'écran qui affiche la fenetre courante
  • L'objet Document : c'est encore le DOM dans son ensemble (d'une manière générale)
